# Havana Home 🇨🇺

A premium, responsive single-page application for a boutique stay in Old Havana. 

## 🚀 Tech Stack
- **React 19** + **Vite**
- **MUI (Material UI) v6**
- **TypeScript** 
- **Swiper.js** 
- **Google Maps API** 


## Getting Started

### Installation

Install the dependencies:

```bash
npm install
```

### Development

Start the development server with HMR:

```bash
npm run dev
```

Your application will be available at `http://localhost:5173`.
